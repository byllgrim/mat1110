$$\frac{df}{dx} = 2x$$
$$\frac{df}{dy} = 2y$$

$$z = f(1,1) + \frac{df}{dx}(1,1)(x-1) + \frac{df}{dy}(1,1)(y-1)$$
$$= (1^2 + 1^2) + (2\cdot 1)(x-1) + (2\cdot 1)(y-1)$$
$$= 2 + 2(x-1) + 2(y-1)$$
